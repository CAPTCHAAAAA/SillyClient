# SillyClient 1.5.0

这次更新把“创建实例”从一次界面操作变成了完整的安装事务，也补齐了 Android 与 Windows 在状态、删除和控制台行为上的差异。

## 更新内容

- 删除实例前增加两阶段确认，并与动态、深色、浅色主题保持一致
- 创建本地实例时展示下载、解压、依赖安装和可运行性检查进度
- 只有 SillyTavern 确认可访问后，实例才会写入列表并显示创建成功
- 安装失败会清理未完成目录，避免残留实例被误识别
- 修复实例卡片、菜单与横向轮播的裁切和异常阴影
- 修复应用重启后实例运行状态可能失真的问题
- 修复关闭酒馆后原生层仍报告 ready 的问题
- 增加实例 ID 路径校验与 zip 解压越界防护
- Windows 端改用真实的 `cmd.exe` 控制台提示，不再显示 Termux 文案
- Windows 内置运行时按生产资源路径加载，安装版可直接运行

## 下载

- `SillyClient-Android-v1.5.0.apk`
- `SillyClient-Windows-v1.5.0.exe`

Android 支持 Android 8.0 及以上的 arm64 设备。Windows 支持 Windows 10 / 11 x64。
