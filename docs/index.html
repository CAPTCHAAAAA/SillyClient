内置运行时

Node.js 随应用打包，开箱即用。不需要 Termux，不需要自行安装任何依赖，下载安装后直接启动。

Android + Windows，同一 React 前端。

液态玻璃界面，状态栏适配，手势。

双平台共享同一套 React 前端。
