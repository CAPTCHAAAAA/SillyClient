Tarven++ bootstrap directory.

This overlay intentionally contains only Tarven++ owned files and empty target directories.
Do not copy SillyDroid generated assets here.
Do not copy Termux app as a shell here.

Future build-stage outputs should be generated by Tarven++ build pipeline:
- server/server-source.zip from upstream SillyTavern source selected by Tarven++
- rootfs/rootfs-fs.zip and rootfs/rootfs-usr.zip from Tarven++ runtime builder
- server/dependency-packs/*.zip from Tarven++ dependency pack builder
